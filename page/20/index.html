go-import

正则文法

ImportDecl       = "import" ( ImportSpec | "(" { ImportSpec ";" } ")" ) .
ImportSpec       = [ "." | PackageName ] ImportPath .
ImportPath       = string_lit .

导入路径(ImportPath)

  • 相对路径
  • 绝对路径(gopath)

包名(PackageName)

  • .

这样导入后,在使用的时候无须包名

  • _

不直接使用该包内的导出标识符

  • 别名

通过别名访问(“f.Println”)

  • 默认形式

参考

  1. https://golang.org/ref/spec#Import_declarations

| 是否方便集中式管理 | 否 | 是 |
