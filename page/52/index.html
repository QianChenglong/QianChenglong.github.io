  • [控制面板] => 【小图标】 => 凭据管理器

  • control userpasswords2 => [高级] => [管理密码]

  • control keymgr.dll

注：

  1. hostname为要连接的电脑主机名，也可以是IP

  • 修改后，注销生效！
