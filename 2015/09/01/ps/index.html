ps

说明

  • 破折号开头的为UNIX风格

  • 不带破折号开头的为BSD风格

  • gnu long option，双破折号开头

选项

  • -e

    同-A完全一样，选择所有进程

  • -f

    输出所有信息

  • -j

    jobs格式，输出PGID SID C STIME

  • -o format(/STANDARD FORMAT SPECIFIERS)

    指定输出格式(列)

    • pid

    • lstart 长格式启动时间

    • cmd 启动命令

    • wchan 内核阻塞

  • L Show threads, possibly with LWP and NLWP columns.

OUTPUT MODIFIERS

  • –forest

    进程树

实例

  • 查看当前用户进程

    ps -f

  • 指定进程启动的时间

    ps -eo pid,lstart,cmd|grep user_manage
